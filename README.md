## How to Contribute
1. Clone the repo: `git clone https://github.com/Naledi-Reed/Naledi-Reed.github.io.git`
2. Check out your branch: `git checkout feature/your-section`
3. Make your changes to `index.html`
4. Commit with a meaningful message: `git commit -m "Add hero animation"`
5. Push: `git push origin feature/your-section`
6. The Project Manager will merge into `main`
